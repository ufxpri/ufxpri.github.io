예산 설정
budget에서 비용 초과 시 알람을 줄 수 있는 시스템이 있다.

EC2 란 AWS에서 제공하는 가상 머신 이다.
다음의 능력을 가지고 있다.
- 가상 장치 대여 (EC2)
- 가상 드라이브에 데이터 저장 (EBS)
- 장치마다 부하 분산 (ELB)
- 자동 크기 조정 그룹을 통해 규모 조정 (ASG)

장치는 window, mac, linux 로 부팅 가능하다.

boot strap 이란 장치가 시작할때 처음 실행하는 스크립트를 의미한다.

7개의 인스턴스 유형이 있다.

m5.2xlarge
m - 인스턴스 클래스
5 - 세대 번호
2xlarge - 인스턴스의 크기

- 일반 사용 용도
- 컴퓨팅 최적화 - c
- 메모리 최적화 - r
- 스토리지 최적화 - i

방화벽을 생각해보자. 방화벽은 매우 기초적이고 필수적인 부분이다.

security group 은 inbound 제약 outbount 허용
여러개의 인스턴스에 연결될 수 있다.
vpc 에 제한되어 있다. 
ssh 엑세스의 경우 별도로 보안그룹을 만들어 관리하는것이 좋다.
연결 거부가 발생한 경우 시큐리티그룹은 성공했는데 어플리케이션에서 응답이 없는 것 이다.
시큐리티 그룹 간의 연결 또한 제어할 수 있다.

22 ssh
21 ftp
22 sftp
80 http
443 https
3389 rdp
