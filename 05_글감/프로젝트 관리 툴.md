# 정리

현제까지 알아본내용은 우리 회사는 github 유료버젼을 사용하고 있고 github를 적극 활용하는게 좋을것으로 보인다. 따라서 slack 과 github 를 사용한다. notion 을 도입하면 조금더 예쁘게 (문서)관리할 수도 있을것 같지만 일단 제쳐둔다.

따라서 내가 정리해야할 내용은 slack 의 100% 활용방법과 github 의 사용방법 및 이슈 관리방법을 정리하면 된다.
